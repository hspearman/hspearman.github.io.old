Portfolio Some projects I've worked on

BandBot www.bandbot.com

  • SOURCE: https://github.com/hspearman/dynamic_bng
  • WHEN: Summer 2015
  • # OF TEAMMATES: Solo

BandBot is a web application that dynamically generates band names based on your favorite artists. It parses a user's Spotify library and intelligently generates a band name based on the results.

Outer Space Canvas

  • SOURCE: assembla.com/code/ectproject/subversion/nodes
  • WHEN: Winter 2014
  • # OF TEAMMATES: 1

Outer Space Canvas is a mock website that sells framed canvases of Hubble space photos. It was built with ASP.NET/C# and interfaces with a database via the Entity framework to store and retrieve site information.

Elevator System Simulator

  • SOURCE: github.com/hspearman/ElevatorSimulator
  • WHEN: Spring 2014
  • # OF TEAMMATES: 1

Elevator System Simulator is a multi-threaded program that simulates a system of elevators within a building. It optimally handles elevator requests and gathers and reports detailed statistics during the simulation.

Swipe Keyboard

  • SOURCE: assembla.com/code/csc394/subversion/nodes
  • WHEN: Winter 2014
  • # OF TEAMMATES: 5

Swipe keyboard is a virtual keyboard that allows users to generate text via gestures. The program features a set of algorithms that analyzes user gestures to produce the best-fit word.

Celebrity Guessing Game

  • SOURCE: N/A
  • WHEN: Winter 2014
  • # OF TEAMMATES: 1

Celebrity Guessing Game is a multi-threaded TCP server that hosts a text-based game which guesses a famous person that the player has in mind. The program dynamically learns and serializes new celebrities from players and adds them to the underlying tree structure.
